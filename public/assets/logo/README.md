# Envert Logo Folder

Upload your logo files here. Supported formats:

- `logo.png` — Full logo with background (recommended: 300x80px)
- `logo.svg` — Vector logo (best quality, scales perfectly)
- `logo-dark.png` — Logo variant for dark backgrounds (optional)
- `logo-light.png` — Logo variant for light backgrounds (optional)
- `logo-icon.png` — Icon-only version (square, e.g. 64x64px)
- `favicon.ico` — Place in the `public/` root for browser tab icon

> The Navbar component will automatically use `logo.svg` or `logo.png` from this folder.
> If no logo is found, the site will show the "Envert" text logo as a fallback.
